    ```
    4. `std::accumulate()`
Используется для вычисления суммы элементов или выполнения любой другой накопительной операции.

*   **Где живет:** `<numeric>` (важно не забыть этот инклюд!)
*   **Синтаксис:** `accumulate(начало, конец, начальное_значение)`.
*   **Пример:**

    std::vector<int> v = {1, 2, 3};
    int sum = std::accumulate(v.begin(), v.end(), 0); // Результат: 6
Внимание:** Тип результата зависит от типа третьего аргумента. Если передать `0.0`, расчет будет в `double`.
    
    ### 5. std::transform()
Применяет заданную операцию к каждому элементу диапазона и записывает результат в другой (или тот же самый) диапазон.

Где живет: <algorithm>
Логика: «Взять элемент $X$, превратить его в $Y$ и сохранить».
Пример: Возведение всех чисел в векторе в квадрат.
    

    std::vector<int> v = {1, 2, 3};
    std::transform(v.begin(), v.end(), v.begin(), [](int x) { 
        return x * x; 
    }); // v теперь {1, 4, 9}
    ```

//transform

Функция принимает четыре (или пять) аргумента:

    Откуда начинаем читать (v.begin()).

    Где заканчиваем читать (v.end()).

    Куда начинаем записывать (v.begin() — в вашем случае).

    Что делать с каждым элементом (лямбда-выражение).
