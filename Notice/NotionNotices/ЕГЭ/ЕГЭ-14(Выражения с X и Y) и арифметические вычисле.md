    ''' 274)  (П.М. Волгин) Значение арифметического выражения
    26**2 + 169 - 11
    
    записали в системе счисления с основанием 13.
    
    В этой записи помимо
    цифр от 0 до 9 могут встречаться цифры из списка: А, B, С==12, которые имеют
    числовые значения от 10 до 12 соответственно. Сколько цифр C и цифр 2
    встречается в этой записи? '''
    
    def kdig(n, p, k):
        cntK = 0
        cnt2 = 0
        s = 0
        while n > 0:
            dig = n % p
            if dig == k:
                cntK += 1
            if dig == 2:
                cnt2 += 1
            n //= p
        s = cntK + cnt2
        return s
