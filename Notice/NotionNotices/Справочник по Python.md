    ## Типы данных в python
    
    ```python
    int - целые числа(1,2,3 ....)
    float - числа с плавающей точкой(1.1, 3.0,2.9 ....)
    str - строки("Hello world")
    list - списки([1 , 2.9 , "hello"])
    dict - словари(ключ + значение)({"john" : "+1-23-45"}
    tuple - списки([1 , 2.9 , "hello"]) ,но нельзя изменить
    set - последовательность уникальных объектов,которые не повторяются
    ({"a", "b"})
    bool - логические значения(true, false)
    
    ```
    
    ## Простые математические действия
    
    ```python
    //В python при делении ,результат всегда будет с плав. точкой
    //Также если сложить(вычесть) целое число с числом с плав.точкой результат
    //будет с плав. точкой
    
    round() - округляет результат после мат. вычеслений
    round(1.2*3, 2) - сначало само выражение, затем до кого числа округляем
    
    3%2 - при такой операции мы получем остаток после деления
    = - оператор присваивания
    == - оператор сравнения
    
    //Если в больших мат.выражения есть деления ,то результат будет 
    с плав. точкой
    2**18 - чтобы возвести 2 в 18 степень
    ```

- ***тип данных String, функция String***
    
    ```python
    "I'm Elias and I'm a \"strong\" programmer" - чтобы внутри строки можно 
    было повторять " и '  
    print(r"C:\USERS\documents") - чтобы вывелись \ корректно
    print("hello \nworld") - \n чтобы перенесьти слова на новую строку
    print("hello \tworld") - \t чтобы оставить отступ от слова
    hello = str("hello world") - чтобы переменной задать значение строки
    
    hello = str("hello world")
    hello[0] - чтобы вывести букву из строки
    hello[-1] - также можно вывести букву конца
    hello[3:8] - чтобы вывести буквы с 4 по 9
    hello[3:] - чтобы вывести все буквы с 4 и до конца
    hello[3:8:2] - чтобы вывести буквы с 4 по 9 и забирая от туда каждую 
    вторую
    "my" + "name"- чтобы склеить строки
    print("{} {}".format(hello,world)- чтобы склеить строки
    'a'*5 = 'aaaaa'
    ```
