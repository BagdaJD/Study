**copy()** - используется для ввода значений в контейнер, для вывода значений
```C++
vector<int> v{1, 2, 3, 4, 5, 6, 9};

//вывод значений
copy(v.begin(), v.end(), ostream_iterator<int>(cout, " "));
//вывод значений
//ostream_iterator<int>(cout, " ")) - указываем тип выводимых данных в <>, в () указываем cout и как мы будем выводить(в данном случае, через " ")

------------------------
//ввод значений
vector<int> v;
copy(istream_iterator<int>(cin), istream_iterator<int>(), back_inserter(v));

//первые два аргумента используются по дефолту, ток указываем в <> тип данных, а последним указываем куда записываем через back_inserter() 
------------------------

```


**copy_if()** - скопировать из одного контейнера в другой c определенными условиями(лямбда-функция)

Указываем сначала - **откуда - до куда**, а затем куда вписываем через **back_inserter(new_v)** 

```C++
vector<int> v{1, 2, 3, 4, 5, 6, 9};
copy(v.begin(), v.end(), ostream_iterator<int>(cout, " "));

vector<int> v1;
copy_if(v.begin(), v.end(), back_inserter(v1), [](int x) {
    return x % 3 != 0;
});
cout << endl;
copy(v1.begin(), v1.end(), ostream_iterator<int>(cout, " "));
```

**accumulate()** - находит сумму также с учетом лямбда-функции
!Лямбду-функцию необязательно использовать, тогда посчитается просто сумма элементов контейнера!
```C++
vector<int> v{1, -10, 9, 5};
copy(istream_iterator<int>(cin), istream_iterator<int>(), back_inserter(v));
//1 -10 9 5
copy(v.begin(), v.end(), ostream_iterator<int>(cout, " "));

int sum = accumulate(v.begin(), v.end(), 0, [](int total, int x) {
	return (abs(x) < 10) ? (total + abs(x)) : total;
});
//Пишем откуда и докуда будем прочитывать элементы контейнера
//0 - обязательно пишем, задавая стартовое значение суммы
//в [] указываем сначала накопительную переменную, вторую переменную указываем как бегунок по элементам
//в конце через return указываем при каких условиях мы суммируем total

```

**v.erase(remove_if(), v.end())** - удаляет в данной контейнере элементы удовлетворяющие условию из лябда-функции 
```C++   
vector<vector<int>> new_matr = {
        {1, 2, 4},
        {0, 0, 0},
           {1, 0, 1}
    };

new_matr.erase(remove_if(new_matr.begin(), new_matr.end(), [](const vector<int>& v) {
        return all_of(v.begin(), v.end(), [](int elem) { return elem == 0; });
    }), new_matr.end());
```


**remove_copy_if()** - копирует элементы в новый контейнер, но удаляет определенные элементы из прошлого контейнера, которые удовлетворяют условию из лябда-функции
```C++
vector<int> v = {1, 2, 3, 4}
vector<int> new_v;
remove_copy_if(v.begin(), v.end(), back_inserter(new_v), [sred_arith](int x) {
        return x < sred_arith;
});
```
