filter{...} - фильтрует коллекцию и получает что-то новое засчет этого

val listOfNumbers = mutableListOf<Int>()
val listOfEvenNumbers = listOfNumbers.filter({number:Int -> number % 2 == 0})  
//Пример работы функции filter({})
В качестве параметров принимает анонимную функцию
Здесь можно опустить указание типа данных для number, т.к мы явно указали, что mutableListOf точно состоит из Int. Далее, т.к у нас в функции испорльзуется только один параметр можно его заменить на it
------
val listOfEvenNumbers = listOfNumbers.filter({it % 2 == 0})

Правило:
Если анонимная функция указывается в качестве последнего параметра, то круглые скобки можно убрать

val listOfEvenNumbers = listOfNumbers.filter{it % 2 == 0}
