### Перестановка цифр в числе
```
Смотрим внимательно на порядок цифр ,который от нас хотят получить
#a1a0.py
n = int(input('Введите двухзначное число')) - ввод числа int + приглашение к вводу

a1 = n // 10 - получение первой цифры (десятки)
a0 = n % 10 - получение последней цифры (единицы)

n = a0 * 10 + a1 - меняем местами десятки и единицы и получаем новое число)
#Единицы умножаем на десять,тем самым мы из них делаем десятки, а уже старые десятки прибавляем в конце, в качестве единиц
print(n)
#a1a0.py(трехзначное)
n = int(input('Введите трехзначное число:  '))
a2 = n // 100 - первая цифра(сотни)
a1 = n // 10 % 10 - вторая цифра(десятки)
a0 = n % 10 - третья цифра(единицы)

n = a0 * 100 + a1 * 10 + a2 - воссоздаем число с конца
print(n)

### Перестановка цифр в числе(отрицательные) и т.д
```
'''
#a2a1a0
Дано целое трехзначное число n = a2a1a0(ai - цифры числа).Ихменить запись
числа на - n = a1a0a2
*Числа могут быть отрицательными.Сохранить знак числа
'''
n = int(input('Введите целое трехзначное число: '))

z = n // abs(n) - сохраняем знак числа
n = abs(n) - берем n в модуль,для дальнейшим арифметических операций

a2 = n // 100 - удаляем сразу две цифры
a1a0 = n % 100 - оставляем две цифры с конца

n = (a1a0 * 10 + a2) * z
print(n)
