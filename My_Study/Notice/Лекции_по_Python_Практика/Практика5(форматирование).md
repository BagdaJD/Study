### Форматирование для e(экспоненциальная форма записи) и обратно
```
Для экспоненциальных чисел
--------------------
elem = 1.463357657e-3
print('%10.2e' %elem)

Выведет - 
1.46e-03

//Мы выделяем для нашего числа 10 ячеек(для цифр, знаков, порядка)
и указываем, кол-во цифр после запятой(в данном случае 2),
также указываем, что у нас экспоненциальная форма числа(e)

-----------------
-----------------

Из вещественных в экспоненциональные 
--------------------
ln = 1.463357657674867
print('%10.2e' %ln)
//Здесь по такому же принципу

Выведет - 
1.46e+00
```
### Форматирование из целого в вещественное(и наоборот)
```
a = 2024
print('%5.2f' %a) - 2024.00 из целого в вещественное(пишем просто букву f)

------------

a = 98.627823
print('%5d' %a) - 98 -- из вещественного в целое(обрубает число после запятой)

------------

a = 637.8372832
print('%5s' %a) - '637.8372832' перевод строку

```
### Сравнение строк и их форматирование
```
language = 'chinese'
print(language == 'chinese')
Выведет - True

-------------

print('chinese' > 'Chinese')#True
сравнивает по значению символов
//ord('c') = 99
//ord('C') = 67 

print(x, y, z, ... sep =' ') 
sep - разделитель
!но если sep = ''! - то отступов вообще не будет

--------------------

print(x, y, z..., end = '; ')
print(x, y, z)
//x y z; x y z - сразу будет переходить

end - то как будет осуществляться переход к следующему принту(строчке по сути)

--------------------

Управляющие последовательности печати
\ - символ новой строки
\\ - обратный слеш (\)
\’ - апостроф или одинарная кавычка
\” - двойные кавычки
\n - перенос на новую строку
\t - табуляция(много пробелов)

```
### Двойные неравенства в Python и диапазоны

(100 <= x) and (x <= 99)

(x >= 100) and (x <= 999) or (x >= -999) and (x <= -100)

(100 <= abs(x)) and (abs(x) <= 999)
-----
